# Dossiers

- data : le jeu de données "KODAK image dataset" téléchargé depuis http://www.cs.albany.edu/~xypan/research/snr/Kodak.html

- small : les images du jeu, fortement compressées avec perte en JPEG2000.

- big : les images du jeu, faiblement compressées avec perte en JPEG2000.


# Folder

- data : the KODAK image dataset downloaded from http://www.cs.albany.edu/~xypan/research/snr/Kodak.html

- small: the dataset images, highly compressed with loss in JPEG2000

- big: the dataset images, slightly compressed with loss in JPEG2000
